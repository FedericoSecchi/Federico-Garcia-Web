# SEO Topic Clusters – fedegarcia.com

Long-term content and SEO architecture for the Fede García sailing coach portfolio.

## Domain & canonical

- **Production URL:** https://www.fedegarcia.com/
- **Canonical:** Use `https://www.fedegarcia.com/` on all pages.

## Content cluster folders

| Folder | Purpose |
|--------|--------|
| `case-studies/` | Regatta and campaign case studies (e.g. Youth Worlds, Weymouth, Nacra 17) |
| `portfolio/` | Visual portfolio and media (photos, videos, regatta highlights) |
| `coaching-methodology/` | How you coach: methodology, drills, progression, mental game |
| `youth-sailing-development/` | Youth sailing development, pathway, and program content |

## Priority topic clusters

1. **High performance sailing coaching** – Pillar: what it is, who it’s for, outcomes. Link to methodology and case studies.
2. **29er sailing coach** – Class-specific content: 29er training, clinics, results. Link to youth development and case studies.
3. **Youth sailing development** – Pathway, age groups, long-term development. Link to methodology and portfolio.
4. **Olympic sailing training** – Nacra 17, Olympic campaigns, high-level preparation. Link to case studies and methodology.

## Future pages to add to sitemap

When you add new pages, add their URLs to `sitemap.xml`, for example:

- `https://www.fedegarcia.com/coaching-methodology/`
- `https://www.fedegarcia.com/case-studies/youth-worlds-2022`
- `https://www.fedegarcia.com/youth-sailing-development/`
- `https://www.fedegarcia.com/portfolio/`

## Image SEO

- Use descriptive, keyword-rich file names (e.g. `fede-garcia-29er-coaching-weymouth.jpg`).
- Every image: descriptive `alt`, `width`/`height`, `loading="lazy"` (except hero above the fold if needed).
- Compress images > 500 KB (e.g. TinyPNG, ImageOptim) for better PageSpeed.
